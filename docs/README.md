## What is it?

`docs-app-for-codemod-utils` provides the documentation site for `codemod-utils`. The app is deployed on https://codemod-utils.netlify.app/.


## Local development

```sh
# Run the app
pnpm start

# Lint files
pnpm lint
pnpm lint:fix

# Run tests
pnpm test
```
